## Overview

JWT Token Inspector is a production-grade security tool for analyzing JWT tokens with comprehensive vulnerability detection, security scoring, and professional reporting capabilities. Built for security professionals and developers.

## Features

- 🔍 **Token Analysis** - Parse and decode JWT headers, payloads, and signatures
- 🛡️ **Security Scoring** - 0-100 scoring system with risk level classification
- 🎯 **Attack Detection** - SQL injection, XSS, command injection, JWT injection patterns
- 🔑 **Signature Verification** - Support for HS256/HS384/HS512, RS256/RS384/RS512, ES256/ES384/ES512
- 📊 **Multiple Outputs** - Text, JSON, and HTML report formats
- ⚠️ **Vulnerability Scanning** - Detect weak algorithms, sensitive data exposure, missing claims
- 📝 **Comprehensive Logging** - Rotating file logs with forensic detail

# Risk Classification

CRITICAL (0-25) - Immediate action required

HIGH (26-50) - Urgent attention needed

MEDIUM (51-75) - Should be addressed

LOW (76-90) - Minor improvements possible

EXCELLENT (91-100) - Exceptionally secure

# Detection Capabilities

Weak algorithm detection (HS256, none)

Sensitive data exposure (passwords, secrets, keys)

Missing security claims (exp, iat, nbf)

Attack pattern detection (SQL injection, XSS, command injection)

Token entropy analysis
